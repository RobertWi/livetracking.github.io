Live Tracking for Data Nerds

livetracking.io is an free and open source live tracking service. It's useful for recording data and performing analytics in real time. You are able to track your time elapsed, distance, speed and elevation in real time. And if you have any ANT+ or Bluetooth sensors, this data can also be transmitted. Optimized for easy use.

Input

With the Android app IpBike you can transfer your data in real time. Deposit your access data in Settings Upload settings livetracking.io. The app supports ANT+ and Bluetooth sensors. You can transfer your speed, heart rate, cadence and power as well as the GPS coordinates.

Output

livetracking.io is very much based on InfluxDB. So you have all the advantages of a very fast time series database. It aims to answer queries in real-time. Each user comes with his own InfluxDB database for live tracking. Only you as an owner can write to the database. You can give other users read rights to your live tracking database. Currently you can save 8 hours of data.

If you make your profile public, anyone can view your data at the following URL: https://my.livetracking.io/public/YOUR-USERNAME

If your profile is not public, your friends can view your data at the following URL: https://my.livetracking.io/watch/YOUR-USERNAME

Because you have your own InfluxDB database, you can also use other evaluation programs. A popular program is e.g. Grafana. Use the URL https://output.livetracking.io and the database name YOUR-USERNAME.

Developers Welcome

You can use livetracking.io as the back end for live tracking. It is very easy to integrate into existing apps. The transfer is done via an HTTP API. You will find more details in the documentation.
