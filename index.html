Live Tracking for Data Nerds

livetracking.io is an free and open source live tracking service. It's useful for recording data and performing analytics in real-time. It is designed to track athletes in real time. In addition, it is very easy to use and can be quickly integrated in existing applications.

Developers Welcome

You can use livetracking.io as the back end for live tracking. It is very easy to integrate into existing apps. The transfer is done via an HTTP API. You will find more details in the documentation.

Features

This application is very much based on InfluxDB. So you have all the advantages of a very fast time series database. It aims to answer queries in real-time. That means every data point is indexed as it comes in and is immediately available in queries that should return in less than 100ms.

Each user comes with his own InfluxDB database. Only you as an owner can write to the database. You can give other users read rights to your live tracking database. Currently you can save 8 hours of data.
